# DBMS
## Something you should know
<font color="137ac9">**Define**</font>：DBMS 是 DataBase Management System 的簡寫。從名字我們即可看出 DBMS 是一種可以管理資料庫的系統。 <br>
**Example**：具代表性的 DBMS 有 Oracle、Microsoft SQL Server、Access、MySQL 及 PostgreSQL 等

## Take a deep look
我們可以將一個個的 file (database) 看成是包裹或貨物。<br>
如果沒有使用 DBMS，我們就必須自行管理這些雜亂的貨物，如圖右上角。<br> 
如果使用 DBMS，使用者的角色就會像一個管理者負責指揮別人搬運查詢整理這些貨物
